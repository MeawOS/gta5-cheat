# Keyboardology

    "F5" - menu
    "Numpad 5" - confirms the option
    "Numpad 0" - return
    "Numpad 8" - function up
    "Numpad 2" - function down
    "Numpad 4" - decrease the value of a given option
    "Numpad 6" - increasing the value of a given option
